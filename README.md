# Frontend Minimal Template

One Main JS File and One Main CSS File.

## Tech Stack

- Package manager
  - npm
- Transpiler
  - babel
    - babel-preset-env
- JS Compresser
  - UglifyJS
- CSS
  - PostCSS
    - autoprefixer
    - postcss-normalize
    - cssnano
- Linter
  - eslint(babel-eslint)
  - stylelint
- Test Runner
  - ava
- Server
  - browser-sync

## Usage

```sh
# Setup
npm install

# Server start
npm start

# Watch JS and CSS
npm run watch

# Build for prod
npm run build
```

See package.json "scripts" field.
